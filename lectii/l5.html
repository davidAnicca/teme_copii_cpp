Aprofundarea structurii while

  1. Se dă următorul cod de c++ :
    2.         
    #include <iostream>
    
    using namespace std;
    
    int main()
    {
        int a = 4;
        int b = 5;
    
        int k = 0;
    
        int i = 1;
        while(i <= a){              //repetă de 4 ori
            int j = 1;
            while(j <= b){                  //repetă de 5 ori
                k++;
                j++;
            }
            i++;
        }
        
        cout << k << endl;
    
        return 0;
    }
    • Instrucțiunile din primul while se repetă de 4 ori.
    • Instrucțiunile din al doilea while se repetă de 5 ori.
    • Ce valoare va avea k la afișare? Justificați.

    • Explicație:
      • De câte ori se repetă instrucțiunile din primul while, al doilea while începe de la început.
        
        
    • Așadar, "schema programului" arată cam așa:
      • 
        
        
          Numim primul while W1
    Numim while-ul din interior W2
    
    W1(1/4)    >--> W1(2/4)   >--> W1(3/4)   >--> W1(4/4)   >-->| 
    ↓         ↗     ↓         ↗    ↓         ↗    ↓         ↗     
    W2(1/5)   ↑     W2(1/5)   ↑    W2(1/5)   ↑    W2(1/5)   ↑     
    ↓         ↑     ↓         ↑    ↓         ↑    ↓         ↑     
    W2(2/5)   ↑     W2(2/5)   ↑    W2(2/5)   ↑    W2(2/5)   ↑     
    ↓         ↑     ↓         ↑    ↓         ↑    ↓         ↑     
    W2(3/5)   ↑     W2(3/5)   ↑    W2(3/5)   ↑    W2(3/5)   ↑     
    ↓         ↑     ↓         ↑    ↓         ↑    ↓         ↑     
    W2(4/5)   ↑     W2(4/5)   ↑    W2(4/5)   ↑    W2(4/5)   ↑     
    ↓        ↗      ↓        ↗     ↓        ↗     ↓        ↗      
    W2(5/5)↗        W2(5/5)↗       W2(5/5)↗       W2(5/5)↗         
        
      
        
        
    • La fiecare W2 k crește cu 1.
      • 
        
    • Cele 5 W2 se repetă de W1 ori (4); adică 4 * 5.
      • 
        
    • Deci k va avea valoarea 20 la final.
      •

  2. Steluțe
    • Se cere crearea următorului pattern în consolă: 
      •             
    1|*
    2|**
    3|***
    4|****
    5|*****
    6|******
     |...
    n|***** ... *** 
            //n steluțe
    • Sunt în total n linii. Pe fiecare linie sunt n steluțe
      •             
    //Exemplu:
    //n = 3
    *
    **
    ***
    
    //n = 4
    *
    **
    ***
    ****
    
    //n = 7
    *
    **
    ***
    ****
    *****
    ******
    *******
    • Pentru asta avem nevoie de două while-uri:
      •     W1 care să se repete de n ori pentru a scrie liniile
        W2 de i ori pentru a scrie steluțele (unde i este linia curentă 1 -> n)
            
    #include <iostream>
    
    using namespace std;
    
    int main()
    {
        int n;
        cout << "Scrie n: ";
        cin >> n;
        
        int i = 1;
        while(i <= n){          //W1
            int j = 1;
            while(j <= i){          //W2
                cout << "*";            //i steluțe          
                j++;
            }
            i++;
            cout << endl;           //linia următoare
        }
        return 0;
    }
