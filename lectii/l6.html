Aplicații cu structura while

  1. Construcția unui număr
    • Pentru a adăuga o cifră la un număr n trebuie să îl înmulțim cu 10 și apoi să adunăm cifra
      •                 
    Exemplu:
    n = 345; //numărul
    c = 4;    //cifra
    n = n * 10; //n <- 3450
    n = n + c;  //n <- 3450 + 4 = 3454
  2. Să se verifice dacă un număr este palindrom
    • Copiem numărul într-o altă variabilă copie
    • Luăm toate cifrele din copie și le adăugăm la un număr nou (care începe de la 0)
    • Numărul respectiv va fi chiar inversul numărului
      •         
    /*
    Exemplu:
    n = 345 //numărul
    copie = 345 
    invers = 0
    
    invers = 5, copie = 34
    invers = 54, copie = 3
    invers = 543, copie = 0
    
    invers reprezintă inversul lui n
    */
    
    #include <iostream>
    
    using namespace std;
    
    int main()
    {
        int n;
        cout << "introduceti nr: ";
        cin >> n;
    
        int copie = n; //facem o copie
        int invers = 0; //inversul numarului
        
        //construim inversul:
        while (copie > 0){
            int c = copie % 10;  //ultima cifră din copie
            invers = invers * 10; //facem loc pt cifră în dreapta
            invers = invers + c; //adăugăm cifra în dreapta
            
            copie = copie / 10; //eliminăm ultima cifră
        }
        
        if (n == invers){        //am făcut copie pentru a nu pierde valoarea lui n
            cout << "DA";
        }else{
            cout << "NU";
        }
        return 0;
    }
  3. Media școlară
    • Media a k numere naturale se face astfel: 
      •             s/k, unde s este suma numerelor
    • În c++ numerele cu virgulă se pot reține în variabile de tip float.
    • Să se scrie un program care tot citește note de la tastatură până când se introduce numărul 0.
    • Programul va calcula media notelor introduse.

    • Pentru rezolvare, facem suma tuturor numerelor introduse și numărăm câte s-au citit într-o variabilă k.
    • La final împărțim suma la k.
    • Pentru a avea notă cu virgulă, folosim suma ca float.
    • Din citire ne oprim dacă s-a citit 0 de la tastatură.

    • Exemplu: 5 6 10 10 10 0 -> 5+6+10+10+10/5 = 8.2
      •         
    #include <iostream>
    
    using namespace std;
    
    int main()
    {
        int n;
        int k = 0;      //numărul de note introduse
        float sum = 0;  //suma numerelor
    
        cout << "introducei nota>>";     //citim prima notă
        cin >> n;
        while(n != 0){          //citim până când e 0
            k++;
            sum = sum + n;
    
            cout << "introducei nota>>";
            cin >> n;                      //citim următoarele note
        }
    
        cout << "media este: " << sum / k;   ///afișăm media
    
        return 0;
    }
  4. Factorial
    • Factorialul unui număr este valoarea care se obține astfel:
      •     n! = 1 * 2 * 3 * 4 * 5 * 6 * ... * n
    • În matematică se notează cu ! (nu are legătură cu notația din c++)
    • Exemplu: 3! = 1 * 2 * 3 = 6
      Exemplu: 7! = 1 * 2 * 3 * 4 * 5 * 6 * 7 = 5040
      Exemplu: 12! = 1 * 2 * ... * 12 = 479001600

    • Nefiind o operație comodă făcută pe hârtie, se cere să se scrie un program care calculează factorialul unui număr
    • Folosim tipul de date unsigned long long pentru rezultat (fiind un număr mare)
    • Unsigned înseamnă că variabila nu reține numere negative, deci avem mai mult loc pentru factorialul nostru
            
    #include <iostream>
    
    using namespace std;
    
    int main()
    {
        int n;
        cout << "scrieti nr>>";
        cin >> n;
    
        unsigned long long factorial = 1;  //factorialul
    
        int i = 1; //numărul cu care înmulțim (1->n)
        while (i <= n){
            factorial = factorial * i;
    
            i++; //trecem la următorul număr
        }
    
        cout << n << "! = "<< factorial;        
    }
