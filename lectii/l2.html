Ce este o variabilă

În programare folosim variabile pentru a putea reține anumite valori (informații) pe parcursul execuției unui program.

Exemplu:

            
    variabila scor reține un număr.
    la început scor <- 0
    
    dacă câștig atunci:
        scor <- scor + 1 (scorul crește cu 1)
    
    dacă pierd atunci:
        scor <- scor - 1 (scorul scade cu 1)

  3. Par sau impar?
    • Exemplul de ora trecută verifica dacă un număr este par sau impar.
    • Un număr este par dacă restul împărțirii la 2 este 0.
    • Dacă restul împărțirii la 2 este 1, numărul este impar.

      • Exemplu:

                  
                5 : 2 = 2 r 1  			5 este impar
                4
                1
                
                4 : 2 = 2 r 0			4 este par
                4
                0
