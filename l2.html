•   Ce este o variabilă?

 

•   în programare folosim variabile pentru a putea reține anumite valori (informații) pe parcursul execuției unui program.

•   exemplu:

 

variabila scor reține un număr.

la început scor <- 0

 

dacă câștig atunci:

     scor <- scor + 1 (scorul crește cu 1)

 

dacă pierd atunci:

     scor <- scor - 1 (scorul scade cu 1)

•   Aplicație 1

 

•   se cere determinarea sumei a două numere naturale citite de la tastatură.

•   exersare individuală pentru diferență, produs și împărțire întreagă

 

#include <iostream>

 

using namespace std;

 

int main()

{

    int numar1;

    int numar2;

    cout<<"dati primul numar"<<endl<<">>";

    cin>>numar1;

    cout<<"dati al doilea numar"<<endl<<">>";

    cin>>numar2;

 

    int suma = numar1 + numar2;

    int diferenta = numar1 - numar2;

    int produs = numar1 * numar2;

    int impartire = numar1 / numar2;

 

    cout<<"suma: "<<suma<<endl;

    cout<<"diferenta: "<<diferenta<<endl;

    cout<<"produsul: "<<produs<<endl;

    cout<<"catul: "<<impartire<<endl;

 

    return 0;

}

•   Aplicație 2

 

•   exemplul de ora trecută verifica dacă un număr este par sau impar.

•   un număr este par dacă restul împărțirii la 2 este 0.

•   dacă restul împărțirii la 2 este 1, numărul este impar.

 

Exemplu:

 

5 : 2 = 2 r 1              5 este impar
