Tema A

  • Ridicarea la putere a unui număr se definește astfel:


    •     Exemplu:
    32 = 3 * 3 = 9
    74 = 7 * 7 * 7 * 7 = 2401
    12023 = 1 * 1 * ... * 1 = 1
    40 = 1 🤯
    λ0 = 1, (∀)λ ∈ ℜ
        (prin convenție orice număr ridicat la puterea 0 este 1)
  • Scrieți un program care primește un număr n și un număr k și care calculează valoarea lui nk.
  • Datele de intrare trebuie să respecte următoarele restricții:
    •     n ≥ 0
    k ≥ 1

Tema B

  • Pentru a genera un număr random în c++ este nevoie să importăm următoarele două biblioteci:
    •             
    #include <cstdlib>
    #include <time.h>
  • Instrucțiunea "***" trebuie introdusă la începutul codului pentru a genera la fiecare rulare alte numere, astfel:
    •             
    #include <iostream>       
    #include <cstdlib>
    #include <time.h>
    using namespace std;
    
    int main()
    {
        srand((unsigned) time(NULL));  //***
        
        //restul codului...
        
        return 0;
    }
  • Pentru a afișa un număr random, folosim instrucțiunea rand()
    •             
    cout << rand() << endl;
  • Scrieți un program care citește de la tastatură un număr n.
  • Programul apoi afișează un tabel (o matrice) cu n linii și n coloane, aflându-se pe fiecare poziție valori de 0 sau de 1 în mod aleatoriu (random)
  • La fiecare rulare a programului se cere să se afișeze o cu totul altă variantă de aranjare.
    •             
    Exemplu:
    n = 5 (rulare 1)
    10111
    11010
    10010
    10011
    01110
    
    n = 5 (rulare 2)
    
    00100
    01100
    11101
    01000
    10011
  • Restul împărțirii la 2 poate fi doar 0 sau 1, astfel, dacă afișăm restul împărțirii la 2 a valorii returnate de rand(), vom obține doar valori de 0 sau 1.
    •             
    cout << rand() % 2;
