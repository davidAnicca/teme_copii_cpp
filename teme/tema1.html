Tema A

  • Creați o aplicație de tip consolă care citește de la tastatură două stringuri nume1 și nume2.
  • Ulterior programul afișează în consolă nume1 + nume2 = ♥.
  • Exemplu: Andrei + Ana = ♥
  • Puteți folosi ca inspirație codul dat la prima lecție în care se citea un nume și se afișa numele respectiv împreună cu un salut:
  • Inimioara este codată în ASCII cu valoarea 3.
  • Următorul exemplu vă arată cum se afișează în consolă o inimioară:
        
    cout << (char) 3 << endl;
  • Rețineți că operatorul << endl; introduce o linie nouă în consolă.
  • Este important să folosim endl doar atunci când este nevoie.

Tema B


  • Accesați site-ul ASCII Art și alegeți de acolo 3 modele pe care să le afișati în consolă.
  • Folosiți funcția prin care compilatorul vă anunță acolo unde s-au identificat erori în cod.
  • Anunțați compilatorul despre caracterele speciale folosind caracterul \ înaintea acestora.
  • Aveți în vedere că nu toate caracterele speciale generează erori de sintaxă. Puteți identifica erorile de afișare în momentul în care modelul dorit nu este afișat corect.
  • Remediați erorile astfel încât în consolă să se afișeze corect desenul ASCII.
  • Mai jos aveți o listă cu câteva caractere speciale pe care C++ le folosește în sintaxă:

Caracter Reprezentarea ASCII Valoarea ASCII Cum trebuie scris în cod ca să nu mai fie special
Backslash \ 92 \\
Semnul întrebării ? 63 \?
Apostrof ' 39 \'
Ghilimele " 34 \"

  • Exemplu: 
    
    cout << "    /\\_____/\\" << endl;
    cout << "   /  o   o  \\" << endl;
    cout << "  ( ==  ^  == )" << endl;
    cout << "   )         (" << endl;
    cout << "  (           )" << endl;
    cout << " ( (  )   (  ) )" << endl;
    cout << "(__(__)___(__)__)" << endl;
