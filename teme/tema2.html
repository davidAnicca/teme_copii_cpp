Înainte de a face tema, consultați lecția pentru inspirație.

Tema A

  • Creați o aplicație de tip consolă care citește de la tastatură două numere întregi nr1 și nr2.
  • Ulterior programul afișează în consolă media lor aritmetică.
  • Media aritmetică a două numere este suma acestora împărțită la 2.
  • Exemplu nr=10; nr2=4; pe ecran se va afișa 7 deoarece (10+4)/2 = 7.

Tema B

  • Creați o aplicație de tip consolă care citește de la tastatură două numere întregi nr1 și nr2.
  • Ulterior programul afișează în consolă diferența pozitivă a acestora.
  • Prin diferența pozitivă se înțelege scăderea din numărul mai mare numărul mai mic.
  • Exemplu nr=10; nr2=4; pe ecran se va afișa 6 deoarece 10 - 4 = 6.
  • Exemplu nr=4; nr2=10; pe ecran se va afișa 6 deoarece 10 - 4 = 6.

Tema C

  • Citiți din consolă un string cuvant
  • Verificați dacă cuvant este palindrom și afișați pe ecran un mesaj corespunzător
  • Un cuvânt este palindrom dacă se citește la fel și în oglindă
  • Exemplu: radar, supus, rotor, capac, cec, alabala ...
  • Nu uitați să importați biblioteca <string>
  • Biblioteca <algorithm> oferă metoda reverse care inversează un string
        
            #include <iostream>
            #include <string>
            #include <algorithm>
            using namespace std;
            
            int main()
            {
                string cuvant = "david";
                reverse(cuvant.begin(), cuvant.end());
                cout << cuvant << endl; 
                return 0;
            }
  • Folosiți corect metoda reverse aplicată pe o copie a cuvântului citit de la tastatură pentru a putea verifica dacă acesta este sau nu palindrom.
